# Storyverse
*A blog dedicated to posting and reading stories.*

Storyblog is a full-stack project with:
- Backend: Spring Boot (Java, Maven)
- Frontend: React, Vite
- Database: MySQL

To run the app locally, the following are needed:
- Java 17+
- Maven 3.8+, or Maven Wrapper
- Node.js 18+
- npm

How to use the app:
1. Go to the backend folder (**cd storyblog**).
2. Build with the following command: **./mvnw clean package**
3. Run command: **./mvnw spring-boot:run**
4. Backend isnow running at address: **http://localhost:8080**.
5. Change to frontend folder (**cd storyblog-frontend**)
6. Install dependencies: **npm install**
7. Run: **npm run dev**
8. Frontend is now running at **http://localhost:5173**
9. Enjoy!
